## Skill Cluster (7)

1. **Wasser- & Wurzelmanagement**  
   Ziel: Wasserverfügbarkeit, Sauerstoff in der Wurzelzone und Substratdynamik balancieren.  
   Subskills: Gießrhythmus, Drain-Interpretation, Root-Zone-Risiko, Umtopf-Timing.

2. **pH- & EC-Logik**  
   Ziel: Messwerte korrekt nutzen, um Nährstoffaufnahme und Salzhaushalt zu stabilisieren.  
   Subskills: pH-Messdisziplin, EC-Trenddeutung, Kalibrierung/Verifizierung, Flush-Entscheidung.

3. **Nährstoffdynamik (Makro)**  
   Ziel: N/P/K-Bedarfe über Entwicklungsphasen und Symptome hinweg richtig justieren.  
   Subskills: Mangel vs. Überschuss unterscheiden, Dosisanpassung, Phasenwechsel-Ernährung.

4. **Sekundärnährstoffe (Ca/Mg)**  
   Ziel: Ca/Mg-typische Symptome erkennen und über Ursache statt nur Symptom handeln.  
   Subskills: Symptom-Lokalisation, Interaktionen mit pH/EC, gezielte Ergänzung.

5. **Klima-Verständnis**  
   Ziel: Temperatur, Feuchte und Luftbewegung in Wirkungsketten auf Stress/Risiko einordnen.  
   Subskills: VPD-nahe Denkweise, Mikroklima-Management, Tag/Nacht-Differenz.

6. **Phasenverständnis**  
   Ziel: Übergänge zwischen Vegetation, Vorblüte und Blüte proaktiv steuern.  
   Subskills: Licht-/Fütterungs-Transitionen, Timing-Strategie, Erwartungsmanagement.

7. **Diagnostisches Denken (Meta)**  
   Ziel: Unsicherheit strukturiert reduzieren statt impulsiv zu reagieren.  
   Subskills: Hypothesenbildung, Priorisierung, Gegenprüfung von Daten, Iterationslernen.

## Event → Skill Mapping

1. **Unterwässerung**  
   Primary: Cluster 1 · Secondary: Cluster 5  
   Learning Goal: Verstehen, wann ein durchdringendes Gießen nötig ist und wie Nachwirkungen zeitverzögert auftreten.
2. **Überwässerung**  
   Primary: Cluster 1 · Secondary: Cluster 5  
   Learning Goal: Erkennen, dass zu viel Wasser auch Sauerstoffmangel bedeutet und Drainageentscheidungen wichtiger als Reflexgießen sind.
3. **pH-Drift vermutet**  
   Primary: Cluster 2 · Secondary: Cluster 7  
   Learning Goal: Diffuse Symptome mit Messung absichern statt blind zu korrigieren.
4. **Leichte Überdüngung**  
   Primary: Cluster 2 · Secondary: Cluster 3  
   Learning Goal: Frühe Überversorgung durch moderate Gegensteuerung entschärfen.
5. **Stickstoffmangel (Vegetation)**  
   Primary: Cluster 3 · Secondary: Cluster 6  
   Learning Goal: Vegetative N-Bedarfe erkennen und Überkorrekturen vermeiden.
6. **Kaliumbedarf steigt (Blütebeginn)**  
   Primary: Cluster 3 · Secondary: Cluster 6  
   Learning Goal: Phasenabhängige Makroverschiebung (mehr K) nachvollziehen.
7. **Hitzeperiode**  
   Primary: Cluster 5 · Secondary: Cluster 1  
   Learning Goal: Hitzestress als Zusammenspiel aus Klima und Wasserhaushalt behandeln.
8. **Sehr hohe Luftfeuchtigkeit**  
   Primary: Cluster 5 · Secondary: Cluster 7  
   Learning Goal: Hohe Luftfeuchte als Risikohebel für Krankheitsdruck verstehen.
9. **Früher Schädlingsbefall**  
   Primary: Cluster 7 · Secondary: Cluster 5  
   Learning Goal: Frühe Signale priorisieren und frühzeitig mit niedriger Eingriffsintensität reagieren.
10. **Salzaufbau im Substrat**  
    Primary: Cluster 2 · Secondary: Cluster 1  
    Learning Goal: Salzakkumulation über Drain/EC erkennen und über Flush-Strategien auflösen.
11. **Umtopfen steht an**  
    Primary: Cluster 1 · Secondary: Cluster 6  
    Learning Goal: Kurzfristigen Umtopfstress gegen mittelfristigen Wachstumsvorteil abwägen.
12. **Lichtzyklus geändert**  
    Primary: Cluster 6 · Secondary: Cluster 5  
    Learning Goal: Verständnis für Rhythmuswechsel und deren kurzfristige Stressfolgen aufbauen.
13. **Blätter rollen sich ein**  
    Primary: Cluster 5 · Secondary: Cluster 7  
    Learning Goal: Blattmorphologie als Frühindikator für Klima-/Wurzelstress deuten.
14. **Magnesium-Mangel**  
    Primary: Cluster 4 · Secondary: Cluster 2  
    Learning Goal: Mg-Symptome von anderen Mängeln differenzieren und gezielt beheben.
15. **Zugluft / starker Ventilator**  
    Primary: Cluster 5 · Secondary: Cluster 7  
    Learning Goal: Luftbewegung als dosisabhängigen Faktor (nützlich vs. schädlich) einordnen.
16. **Regenperiode (Outdoor)**  
    Primary: Cluster 1 · Secondary: Cluster 5  
    Learning Goal: Outdoor-Risiken über Prävention (Regenschutz/Drain) steuern.
17. **Wurzelzone zu kalt**  
    Primary: Cluster 5 · Secondary: Cluster 1  
    Learning Goal: Wurzeltemperatur als limitierenden Faktor für Aufnahme/Growth begreifen.
18. **Calcium-Mangel**  
    Primary: Cluster 4 · Secondary: Cluster 2  
    Learning Goal: Ca-Probleme als Versorgungs- und Transportthema verstehen, nicht nur als Düngemenge.
19. **Blattfarbe sehr dunkel (N-Überschuss)**  
    Primary: Cluster 3 · Secondary: Cluster 2  
    Learning Goal: Überschusssymptome früh deuten und Gegenmaßnahmen in kleiner Dosis umsetzen.
20. **Blüte stagniert**  
    Primary: Cluster 6 · Secondary: Cluster 7  
    Learning Goal: Stagnation systemisch analysieren statt einzelne Werte isoliert zu optimieren.
21. **pH im Gießwasser nicht gemessen**  
    Primary: Cluster 2 · Secondary: Cluster 7  
    Learning Goal: Messdisziplin als Kernkompetenz für reproduzierbare Entscheidungen verankern.
22. **Trauermücken**  
    Primary: Cluster 1 · Secondary: Cluster 5  
    Learning Goal: Oberflächenfeuchte und Schädlingsdruck zusammen denken und präventiv handeln.
23. **Nachtabsenkung fehlt**  
    Primary: Cluster 5 · Secondary: Cluster 6  
    Learning Goal: Tag-/Nacht-Differenz als Taktgeber für gesunden Metabolismus verstehen.
24. **Wachstumsphase endet / Vorblüte**  
    Primary: Cluster 6 · Secondary: Cluster 3  
    Learning Goal: Übergangssignale erkennen und Nährstoff-/Lichtstrategie vorausschauend umstellen.
25. **EC steigt trotz gleicher Düngung (Drain höher)**  
    Primary: Cluster 2 · Secondary: Cluster 1  
    Learning Goal: Trends statt Einzelwerte interpretieren und Messanomalien sauber verifizieren.
